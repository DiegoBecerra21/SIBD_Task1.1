# Tarea 1.1 - Sistemas de Big Data

Este repositorio contiene la **Tarea 1.1** de la asignatura Sistemas de Big Data.  

Se han realizado pruebas y análisis aplicando diferentes funciones sobre el dataset **`titanic_dataset.csv`**, explorando y manipulando los datos para extraer información relevante sobre los pasajeros del Titanic.  

## Contenido del repositorio

- `titanic_dataset.csv` - Dataset con la información del Titanic.  
- `Titanic_Pandas_Practice_Notebook.ipynb` - Código con las funciones y análisis realizados.
